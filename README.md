
# Machine Test MERN Developer



## Run locally

Clone the project

```bash
git clone https://github.com/harshak0007/Machine-Test.git
```

Go to the project directory
### Run Frontend

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

### Run Backend
```bash
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
